## 📋 О ПРОЕКТЕ

DEDSEC LOCKER BUILDER — это мощный конструктор локеров для Windows и Android с:

- 🎨 **Крутым хакерским GUI** в стиле DedSec (матричный дождь, ASCII-черепа, анимации)
- 💀 **Полноэкранным локером** с требованием выкупа
- 🔑 **Кодом разблокировки** `D3DS3C` (можно сменить)
- 🤖 **Двойным управлением** через Discord + Telegram одновременно
- 🕵️ **Глубоким деаноном** жертвы (IP, гео, система, MAC)
- ⌨️ **Кейлоггером** (в памяти, без файлов)
- 📸 **Скриншотером** экрана
- 🔐 **Стилером паролей** Chrome/Edge
- 🌐 **OSINT-инструментами** (WHOIS, DNS, IP-info, порт-скан, email, телефон, username)
- 💣 **Ботнетом** (DDoS, майнинг)
- 📦 **Отправкой вредоносных пакетов** (TCP-флуд, self-stress)
- 🧹 **Самоуничтожением** при разблокировке (очистка MBR, реестра, удаление себя)
- 📱 **Сборкой Android APK** с теми же функциями
- 💾 **Сборкой самого билдера в EXE**
